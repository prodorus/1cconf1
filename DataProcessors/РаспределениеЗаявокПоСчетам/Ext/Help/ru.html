Обработка «Распределение заявок по счетам»

Обработка предоставляет возможность вручную или автоматически выполнить распределение Заявок на расходования денежных средств в рамках исполнения государственного контракта.

Суммы заявок на расходование денежных средств, требующих оплаты, могут быть распределены для оплаты, как с отдельных счетов в рамках выполнения государственного оборонного заказа, так и с обычных банковских счетов предприятия автоматически по одному из представленных в системе правил распределения платежей.

В документе Заявка на расходование денежных средств на закладке Оплата за счет средств можно установить процентное соотношение платежа по заявке за счет собственных средств. При этом автоматически рассчитаются сумма оплаты за счет собственных средств и остаток суммы платежа по заявке, который будет распределен между контрактами с заказчиками и, при необходимости, собственными средствами.

Распределение выполняется в рамках специализированного рабочего места Распределение заявок по счетам, обеспечивающего оперативный контроль состояния постатейных лимитов для оплат в разрезе контрактов с заказчиками и наличия доступных денежных средств на банковских счетах.

Слева в табличной части обработки выводятся данные по всем созданным и не оплаченным заявкам на расходование денежных средств, в т.ч. колонки:

  • Номер - номер заявки
  • Получатель
  • Срок оплаты
  • Сумма
  • Остаток к расп. - сумма за вычетом суммы оплаты за счет собственных средств
  • Назначение платежа

Справа в табличной части выводятся следующие колонки:

  • Счет, контракт - номера банковских счетов, между которыми требуется распределить остаток суммы оплаты по заявке
  • План. остаток - остаток на банковском счете за вычетом сумм распределенных заявок
  • Доступ. поступл. - данные вводятся в форме Контракт с заказчиком на закладке График оплат
  • Ост. по статье - данные вводятся в соответствии с Контрактом с заказчиком, вид контракта – Контракт с участником кооперации по ГОЗ, на закладке Калькуляция
  • % - доля суммы заявки в процентном соотношении, подлежащей оплате с данного банковского счета, по отношению к общей сумме заявки
  • Сумма заявки


Распределить заявки по счетам можно вручную или автоматически при помощи кнопки Распределить по счетам. Результат распределения появляется в документе Заявка на расходование денежных средств на закладке Размещение.
