Обработка Сопоставление организаций ЕГАИС

Сервис Сопоставление организаций ЕГАИС предназначен для сопоставления организации из ЕГАИС и контрагента.

Перед сопоставлением необходимо заполнить сведения об организациях ЕГАИС в классификаторе организаций ЕГАИС. Для этого в сервисе сопоставления необходимо выполнить команду Классификаторы ЕГАИС - Загрузить организацию или в Классификаторе организаций ЕГАИС выполнить команду ЕГАИС - Запрос организаций. В результате этого действия откроется окно, в которое необходимо ввести ИНН нужной организации. Автоматическая обработка ответов ЕГАИС настраивается в Настройке параметров учета, вручную обработать ответы можно из формы регистра сведений Запросы ЕГАИС, по кнопке меню Обработать ответы. 

Если в существующем списке контрагентов отсутствует подходящий контрагент для сопоставления организаций, закруженных из ЕГАИС, то новых контрагентов можно создать по кнопке Создать контрагентов. Контрагенты создаются в соответствии с выделенными строками организаций из ЕГАИС.
Двойным щелчком мыши по выбранной организации можно просмотреть сопоставленные записи.
Отмена сопоставления организации из ЕГАИС и контрагента выполняется по кнопке Отменить связь.

Сервис доступен если в Настройке параметров учета включен учет алкогольной продукции.
