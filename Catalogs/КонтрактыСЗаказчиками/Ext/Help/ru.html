Справочник Контракты с заказчиками (покупателями)

Справочник предназначен для хранения контрактов, заключенных с заказчиком, в рамках исполнения государственного оборонного заказа.

Справочник доступен в случае если организация участвует в составе кооперации в рамках исполнения государственного контракта (в Настройке параметров учета установлен флажок Использовать ведение учета платежей по государственным контрактам).

Порядок заполнения контракта с заказчиком.

На закладке Основное:

  • Исполнитель - указывается организация, которая будет выполнять отгрузку или оказание услуг в рамках государственного контракта;
  • Заказчик - указывается контрагент (покупатель), которому будет осуществляться отгрузка или оказание услуг в рамках государственного контракта;
  • Реквизиты контракта - указываются:
    • Предмет - заполняется произвольным текстом отражающим суть операции;
    • "№ договора", "от", "Сумма контракта (руб.)", "Выполняется с";- заполняются данными из контракта
  • Государственный контракт - указывается государственный контракт, в рамках которого оформляется контракт с заказчиком;
  • Банковский счет - указывается банковский счет (уполномоченного банка) организации. Банковский счет должен соответствовать указанному государственному контракту.
  • Договор - указывается договор организации с заказчиков.

На закладке Калькуляция вводятся данные по смете расходов по статьям калькуляции, заполняются из одноименного справочника Статьи калькуляции. При установке флажка Смета частично израсходована добавляется колонка Израсходовано, в которой указываются израсходованные средства для каждой статьи калькуляции.

На закладке График поступления ДС задается график по планируемым поступлениям денежных средств от заказчика.

Контракт с заказчиком можно создать на основании элемента справочника Государственные контракты или из формы элемента справочника Договоры контрагентов по гиперссылке "Ввести контракт".
