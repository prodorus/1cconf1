Справочник Контракты с исполнителями (поставщиками)

Справочник предназначен для хранения контрактов, заключенных с исполнителем, в рамках исполнения государственного оборонного заказа.

Справочник доступен в случае если организация участвует в составе кооперации в рамках исполнения государственного контракта (в Настройке параметров учета установлен флажок Использовать ведение учета платежей по государственным контрактам).

Порядок заполнения контракта с исполнителем.

На закладке Основное:

  • Заказчик - указывается наша организация;
  • Исполнитель - указывается контрагент (поставщик), который будет осуществлять отгрузку или оказание услуг нашей организации в рамках государственного контракта;
  • Устанавливается вид контракта
    • Контракт с участником кооперации по ГОЗ - списание денежных средств с отдельного счета заказчика только на отдельный счет исполнителя
      • Реквизиты контракта - указываются:
        • Предмет - заполняется произвольным текстом отражающим суть операции;
        • "№ договора", "от" - заполняются данными из контракта;
      • Государственный контракт - указывается государственный контракт, в рамках которого оформляется контракт с исполнителем;
      • Банковский счет - указывается отдельный банковский счет (уполномоченного банка) исполнителя.
    • Прочий контракт, платежи с отдельных счетов или собственных счетов - списание денежных списание денежных средств с отдельного или собственного счета заказчика на собственный счет исполнителя.
      • Реквизиты контракта - указываются:
        • Предмет - заполняется произвольным текстом отражающим суть операции;
        • "№ договора", "от" - заполняются данными из контракта;
      • Банковский счет - указывается собственный банковский счет исполнителя.
  • Тип платежа 275-ФЗ - выбирается тип платежа.
  • Договор - указывается договор нашей организации с исполнителем.

В случае вида контракта Контракт с участником кооперации по ГОЗ отображается закладка Сведения о кооперации, на которой указываются контракты с заказчиками, входящие в кооперацию в рамках поставки продукции или оказания услуг по государственному контракту.

На закладке График платежей/виды платежей задается график по планируемым поступлениям денежных средств исполнителю. Каждый этап графика заполняется в соответствии с пунктами контракта.

На закладке Подтверждающие документы указываются виды документы, которые будут передаваться в банк для контроля платежей.

Контракт с исполнителем можно создать на основании элемента справочника Государственные контракты или из формы элемента справочника Договоры контрагентов по гиперссылке "Ввести контракт".
