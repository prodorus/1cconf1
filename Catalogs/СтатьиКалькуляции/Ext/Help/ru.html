Справочник Статьи калькуляции

Справочник предназначен для хранения статей калькуляции, которые используются для отражения сметы контракта с заказчиком.
По умолчанию справочник заполнен предопределенными данными из перечня, рекомендованного в Приказе от 23 августа 2006 № 200 «Об утверждении порядка определения состава затрат на производство продукции оборонного назначения, поставляемой по государственному заказу».
Список статей калькуляции является пополняемым.

Справочник доступен в случае если организация участвует в составе кооперации в рамках исполнения государственного контракта (в Настройке параметров учета установлен флажок Использовать ведение учета платежей по государственным контрактам).
